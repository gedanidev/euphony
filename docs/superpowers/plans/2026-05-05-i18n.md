# i18n Multi-idioma Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add ES/EN/JA language support with a language selector in Settings, using react-i18next with localStorage persistence.

**Architecture:** Install `i18next` + `react-i18next`, create locale JSON files under `frontend/src/i18n/locales/`, initialize i18next in `main.jsx`, then replace hardcoded strings in Layout and all main pages with `t('key')` calls.

**Tech Stack:** react-i18next 14.x, i18next 23.x, localStorage for persistence.

### Task 1: Instalar dependencias y crear configuración i18n

**Files:**
- Modify: `frontend/package.json`
- Create: `frontend/src/i18n/index.js`
- Modify: `frontend/src/main.jsx`

- [ ] **Step 1: Instalar dependencias**

Desde la carpeta `frontend/` (o dentro del contenedor Docker si está corriendo):
```bash
cd frontend
npm install i18next react-i18next
```

Verificar que `package.json` ahora incluye en `dependencies`:
```json
"i18next": "^23.x.x",
"react-i18next": "^14.x.x"
```

- [ ] **Step 2: Crear `frontend/src/i18n/index.js`**

```js
import i18n from 'i18next'
import { initReactI18next } from 'react-i18next'

import es from './locales/es.json'
import en from './locales/en.json'
import ja from './locales/ja.json'

i18n
  .use(initReactI18next)
  .init({
    resources: {
      es: { translation: es },
      en: { translation: en },
      ja: { translation: ja },
    },
    lng: localStorage.getItem('euphony-language') || 'es',
    fallbackLng: 'es',
    interpolation: {
      escapeValue: false, // React already escapes values
    },
  })

export default i18n
```

- [ ] **Step 3: Importar i18n en `frontend/src/main.jsx`**

Reemplazar el contenido completo del archivo:
```jsx
import React from 'react'
import ReactDOM from 'react-dom/client'
import { BrowserRouter } from 'react-router-dom'
import App from './App'
import './index.css'
import './i18n/index.js'

ReactDOM.createRoot(document.getElementById('root')).render(
  <React.StrictMode>
    <BrowserRouter>
      <App />
    </BrowserRouter>
  </React.StrictMode>
)
```
